Multiple of 3

利用位數累加的方式，可以推算出 mod 值。

0xABCD mod 3 = (A 0xFFF ＋ B 0xFF + C * 0xF) mod 3 = (A + B + C) mod 3

最後計算出餘數再直接判斷。
